# jz's Blog

This is the source for my site at [jizhang495.github.io](https://jizhang495.github.io), built with [Astro](https://github.com/withastro/astro).

## Development

```sh
npm install
npm run dev -- --host   # start dev server on 0.0.0.0:4321
npm run build           # build to dist/
npm run preview         # preview the production build
```
